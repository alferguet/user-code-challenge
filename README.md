# User Code Challenge

Code challenge done with Vue and Typescript

## Environment variables

### Stored in .env.production

"VUE_APP_API_URL"

## Project setup

```bash
npm install
```

### Compiles and hot-reloads for development

```bash
npm run serve
```

### Compiles and minifies for production

```bash
npm run build
```

### Lints and fixes files

```bash
npm run lint
```
